## ⚡ Quick Setup

### Step 1: Create the Repository

1. Go to **GitHub → New Repository**
2. Set repository name to exactly: **`kavidu-jayakodi`** (must match your username)
3. Make it **Public**
4. **Do NOT** initialize with README (you'll push one)

### Step 2: Push This Code

```bash
cd kavidu-jayakodi
git init
git add .
git commit -m "✨ Initial commit: Premium GitHub profile"
git branch -M main
git remote add origin https://github.com/Kavindu-2004/kavidu-jayakodi.git
git push -u origin main
```

### Step 3: Enable GitHub Actions

1. Go to **Settings → Actions → General**
2. Set **Workflow permissions** to **Read and write permissions**
3. Click **Save**

### Step 4: Run the Snake Workflow

1. Go to **Actions → Generate Snake Animation**
2. Click **Run workflow → Run workflow**
3. Wait ~1 minute for it to complete
4. An `output` branch will be created with the SVG files

### Step 5: Verify Everything Works

Visit: `https://github.com/Kavindu-2004`

## 📊 How the Snake Works

The snake animation is generated from your **actual contribution graph**. GitHub Actions runs twice daily, reads your contribution data, and generates an SVG of a snake eating your green squares. The animation is stored in the `output` branch.
